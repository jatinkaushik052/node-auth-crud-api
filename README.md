# Node.js CRUD API with JWT Authentication (Express + MongoDB)

A simple and secure backend API built using **Node.js**, **Express**, **MongoDB**, and **JWT Authentication**.  
This project includes:

- User Registration & Login  
- JWT-based Authentication  
- Protected Routes  
- CRUD operations  
- MongoDB (Mongoose) integration  
- Clean folder structure
